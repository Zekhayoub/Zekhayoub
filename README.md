## Approach

I build my projects around reusable building blocks — ingestion, feature engineering, regime detection, anomaly detection, interpretability. This foundation adapts to any domain: the same pipeline logic that monitors energy markets can detect regimes in credit spreads or optimize industrial processes.

Each block is designed to evolve: as I learn new techniques, they plug into the existing foundation without rebuilding from scratch. The goal is always the same — translate a business problem into an analytical challenge, solve it with rigorous ML,  and deliver results that drive decisions.

## Featured Projects

### Data Science & Analytics

- **[Energy Market Monitor](https://github.com/Zekhayoub/energy-market-monitor)** — Modular pipeline that helps industrials optimize electricity procurement by detecting market regimes (HMM), identifying price anomalies (Isolation Forest), predicting price spikes (XGBoost), and explaining the drivers behind each state (SHAP). Built on European ENTSO-E data. Python · scikit-learn · hmmlearn · XGBoost · SHAP.

- **[Crypto Volatility Analyzer](https://github.com/Zekhayoub/crypto-volatility-analyzer)** — Market regime detection and funding rate arbitrage for digital assets. HMM identifies 3 market states (risk-on, risk-off, crisis) while a backtest simulates realistic PnL with execution costs. Python · Rust (PyO3) · Polars.

- **[Credit Spread Monitor](https://github.com/Zekhayoub/credit-spread-monitor)** — Regime detection and stress testing for corporate bond markets. HMM classifies risk states across 20+ years of spread data, with automated Excel reporting for weekly briefings. Python · FRED API · hmmlearn.

- **[Ecamania Esports Analytics](https://github.com/Zekhayoub/ecamania-lol-prediction)** — Role-specific win prediction for professional League of Legends (374K records, ROC-AUC > 0.90). SHAP confirms distinct performance drivers per role. Bayesian ranking corrects small-sample bias for player selection. Python · scikit-learn · SHAP.

### AI & Agents

- **[MindCare Agent](https://github.com/Zekhayoub/mindcare-agent)** — Hybrid AI agent that routes emotional support queries between local ML and cloud LLM via a multi-signal scoring system, reducing API calls by ~80% while maintaining a 5/5 safety score. Python · LangGraph · LangChain · FAISS · MLflow · Docker.

### Systems & Engineering

- **[GPU Cloth Simulation](https://github.com/Zekhayoub/gpu-cloth-simulation)** — Real-time physics simulation of cloth, fully computed on GPU via Compute Shaders. Rust · wgpu · WGSL.

- **[TCLab Control Systems](https://github.com/Zekhayoub/tclab-control)** — Discrete PID controller with feedforward compensation, deployed and validated on physical Arduino hardware. Python · SciPy.

- **[Human Defense](https://github.com/Zekhayoub/TowerDefense)** — Tower Defense game built from scratch with full OOP architecture (9 turret types, 8 enemy types, 9 maps). Java · JavaFX.
